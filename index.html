Mauriçius Nativitatem

Mauriçius Nativitatem is a rhythm game inspired by Patapon (the player is given patterns that have to be repeated in order to succeed). In this game, you will have to help Mauriçius, our fishy protagonist, in order to be able to perform the dark ritual he needs so that he can bring his son to life.

• Developed as part of the JamOn 2021, with "Birth" as theme

• Awarded as the 2nd Best Game in the JamOn 2021

• Built with Unity and C#

Auditory

Auditory is an unique experimental scape-room game project which players will have relaxing moments by experiencing the stimulation of sound and melodies. The key to solve the different puzzles is using the microphone and learning to recognice music patterns, rythms or pitches.

• Developed in 2019 during my erasmus in Norway, along other students from the Nord University of Steinkjer.

• Subsequently modified to make the game work with FMOD.

• Built with Unity and C#

TeFeGe

TeFeGe is an online multiplayer shooter, with a Top Down perspective, in which the player will be able to choose from 5 different characters, to be paired with an opponent in a 3 round duel. The game was part of a project in Matchmaking Research, so most of the focus was put into measuring the abilities of the player, based on the performance shown, and the consequent ranking and matchmaking of the players.

• Developed as part of the Final Degree Project subject of the Degree in Videogame Development at Universidad Complutense de Madrid, focusing on Matchmaking

• Built with Unity and C#, as well as Mirror for the server side

Motor Casa Paco

Motor Casa Paco is a 3D Game Engine built with C++ in a 4-month span. Thanks to its component-based architecture and data-driven structure, once a basic project core is established, it is easy to add more content to the project.

• Developed as part of the Projects 3 subject of the Degree in Videogame Development at Universidad Complutense de Madrid

• Built with Ogre3D (Core), Bullet Physics (Physics), FMOD (Sound), CEGUI (UI) and C++

Vaya Canicastañazos is a 3D platform game in which the player will be controlling a marble through 16 different levels, with 3 different collectables each. Inspired by Monkey Ball, you will need to be careful not to fall from the level to avoid unnecesary restarts.
